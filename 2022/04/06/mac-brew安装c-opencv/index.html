安装步骤

直接用 brew 安装 opencv

brew install opencv

默认路径在 ==/usr/local/Cellar/==，实际文件也在这里。
同时系统会在 ==/usr/local/include== 下面做软连接。在这里下面也能看到 opencv4 的目录。

用这个命令是看不到的。

pkg-config --cflags --libs opencv

因为不是 opencv 了，而是 opencv4。
真正的命令是

pkg-config --cflags --libs opencv4

使用 ==g++== 编译

g++ --std=c++11  -O3 -I./ main.cpp pkg-config --cflags --libs opencv4 -o main
